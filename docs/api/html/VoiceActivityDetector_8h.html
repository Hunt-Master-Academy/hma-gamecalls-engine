Detailed Description

Voice Activity Detection for wildlife call analysis.

This file provides the VoiceActivityDetector class which implements robust voice activity detection specifically tuned for wildlife calls. The detector identifies audio segments containing meaningful acoustic activity and filters out background noise and silence periods.
