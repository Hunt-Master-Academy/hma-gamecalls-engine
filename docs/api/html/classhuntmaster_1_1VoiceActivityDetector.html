Detailed Description

Voice Activity Detection optimized for wildlife call analysis.

The VoiceActivityDetector implements energy-based voice activity detection with adaptive thresholding and temporal filtering. It's specifically tuned for wildlife calls, which often have different characteristics than human speech in terms of frequency content and temporal patterns.

Key features:

  • Energy-based detection with configurable thresholds
  • Temporal filtering to reduce false triggers
  • Pre and post buffering for complete call capture
  • Real-time processing with minimal latency
  • Robust handling of background noise
