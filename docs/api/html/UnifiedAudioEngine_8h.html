Detailed Description

Unified Audio Engine API - Core interface for the Huntmaster audio analysis system.

This file contains the main API for the Huntmaster Audio Engine, providing a unified interface for audio recording, playback, analysis, and real-time wildlife call comparison. The engine is designed to be cross-platform and can be integrated into desktop, mobile, and web applications.
