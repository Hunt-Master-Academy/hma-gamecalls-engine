Detailed Description

Voice Activity Detection configuration parameters.

Controls the behavior of the Voice Activity Detection (VAD) system, which automatically identifies periods of actual wildlife calls versus silence or background noise. Proper tuning improves analysis accuracy and reduces false positives.
