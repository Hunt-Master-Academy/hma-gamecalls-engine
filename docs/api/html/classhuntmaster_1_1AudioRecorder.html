Detailed Description

Real-time audio recording interface optimized for wildlife call capture.

The AudioRecorder class provides a high-level interface for capturing audio from input devices with minimal latency and optimal settings for wildlife call analysis. It supports mono recording at various sample rates and provides real-time audio level monitoring.

Key features:

  • Real-time audio capture with configurable parameters
  • Live audio level monitoring for UI feedback
  • Automatic silence trimming for clean recordings
  • WAV file export with proper formatting
  • Thread-safe recording operations
