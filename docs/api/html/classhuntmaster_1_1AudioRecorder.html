Detailed Description

Real-time audio recording interface optimized for wildlife call capture.

The AudioRecorder class provides a high-level interface for capturing audio from input devices with minimal latency and optimal settings for wildlife call analysis. It supports both file-based and memory-based recording modes, along with real-time audio level monitoring.

Key features:

  • Real-time audio capture with configurable parameters
  • Live audio level monitoring for UI feedback
  • Automatic silence trimming for clean recordings
  • WAV file export with proper formatting
  • Memory-based recording for direct data access
  • Thread-safe recording operations
  • Hybrid recording mode (simultaneous file and memory recording)
